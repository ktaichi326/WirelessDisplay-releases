# WirelessDisplay — releases

Public distribution for **WirelessDisplay** (the source code lives in a private repo).

## Install on iPhone/iPad via AltStore
Add this **source** in AltStore:

```
https://raw.githubusercontent.com/ktaichi326/WirelessDisplay-releases/main/apps.json
```

Then install **Wireless Display** from it. Updates appear automatically when a new version is published here. Requires iOS/iPadOS 26+.

The macOS sender app is attached to each release as `WirelessDisplay-mac.zip`.
